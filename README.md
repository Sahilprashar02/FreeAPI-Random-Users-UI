## 🌟 Overview

The **Global Citizens** web application fetches user profiles from the FreeAPI Random Users endpoint and displays them in a stunning, responsive, dark-themed grid layout. Built with React and Vanilla CSS, it showcases dynamic animations, glassmorphism aesthetics, and clean UI/UX principles.

## 🚀 Features

- **Premium Design:** Dark mode, blurred background gradients, and sleek cards.
- **Responsive Layout:** Adjusts perfectly to all screen sizes (mobile, tablet, desktop).
- **Pagination:** Navigate through hundreds of profiles seamlessly.
- **Detailed Profiles:** View each user's nationality, email, location, phone, and registration details.
- **Error Handling:** Graceful fallback UI for API or network errors.

## 🛠️ Technology Stack

- **React.js 18**
- **Vite**
- **Vanilla CSS3** (No Tailwind)
- **Lucide Icons**
- **FreeAPI**

## 🌐 API Reference

Data is sourced from the [FreeAPI Random Users Endpoint](https://api.freeapi.app/api/v1/public/randomusers).

## 💻 Getting Started

To run the project locally:

```bash
# Clone the repository
git clone https://github.com/yourusername/random-users-ui.git

# Navigate into the project
cd random-users-ui

# Install dependencies
npm install

# Start the development server
npm run dev
```
